# LetsSplit!

## Description

LetsSplit! is a web application that helps users split shopping bills fairly among multiple participants. It automatically calculates the total bill, applies flat or percentage discounts, and distributes the discount proportionally based on each participant's purchase amount. The application also provides a visual representation of expenses using a pie chart.

## Features

- Add and remove participants
- Automatic bill calculation
- Flat and percentage discount support
- Proportional discount allocation
- Interactive pie chart
- Export bill summary as PDF
- Responsive user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Chart.js
- jsPDF

## How to Run

1. Download or clone the repository.
2. Open `index.html` in any modern web browser.
3. Start adding participants and their purchase amounts.
4. Enter the discount details and click **Calculate**.
